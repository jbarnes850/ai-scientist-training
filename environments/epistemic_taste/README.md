# Epistemic Taste

Synthetic multi-turn Bayesian environment for testing:

1. experiment selection under uncertainty
2. belief revision under conflicting evidence

The agent sees a research question, three hypotheses, a prior, five candidate experiments, and a budget of three evidence turns. The environment hides the true hypothesis, likelihood tables, and exact posterior updates.

Tool actions:

- `run_experiment(experiment_id)`
- `report_belief(belief={"H1": ..., "H2": ..., "H3": ...}, stop=true|false)`

## Overview

- Environment ID: `epistemic-taste`
- Type: multi-turn tool environment
- Hypotheses: `3`
- Candidate experiments: `5`
- Evidence budget: `3`
- Modes: `70%` active, `30%` passive

## Frozen Data

The frozen dataset lives in `epistemic_taste/data/`:

- `train.jsonl`: `4000`
- `dev.jsonl`: `500`
- `test.jsonl`: `500`

It is generated by `/Users/jarrodbarnes/ai-scientist-training/scripts/generate_epistemic_taste_dataset.py`.

## Environment Args

| Arg | Type | Default | Description |
| --- | ---- | ------- | ----------- |
| `split` | `str` | `"train"` | Dataset split to load |
| `max_examples` | `int` | `-1` | Optional cap on loaded examples |
| `seed` | `int` | `0` | Shuffle seed for dataset order |
| `trajectory_dump_path` | `str \| None` | `None` | Optional path to write finished trajectory JSONL rows |

## Metrics

The environment logs separate reward components for hosted GRPO analysis:

- `reward`
- `mean_experiment_reward`
- `mean_calibration_reward`
- `final_map_bonus`
- `extra_turn_penalty_total`
- `invalid_belief_penalty_total`
- `malformed_action_penalty_total`
- `mean_regret`
- `mean_brier`
- `final_map_correct`

## Quickstart

Install locally:

```bash
cd /Users/jarrodbarnes/ai-scientist-training
prime env install epistemic-taste -p ./environments
```

Run a smoke eval:

```bash
cd /Users/jarrodbarnes/ai-scientist-training
prime eval run epistemic-taste \
  --env-dir-path ./environments \
  --env-args '{"split":"dev","max_examples":1}' \
  --provider prime \
  --model 'Qwen/Qwen3-30B-A3B-Instruct-2507' \
  --num-examples 1 \
  --rollouts-per-example 1 \
  --max-concurrent 1 \
  --sampling-args '{"temperature":0.3,"max_tokens":1024}' \
  --state-columns episode_summary,trajectory_log,posterior_trace \
  --save-results
```
